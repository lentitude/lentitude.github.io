JAVA 语言中不仅在正则表达式中对字符进行转义操作, 而且在字符串中还需要再进行一次转义。如果没有充分理解其中的转义规则, 进行相关操作时会觉得逻辑混乱。这篇文章主要介绍在 JAVA编程中需要注意的转义操作。

为什么要转义

字符的转义主要是为了避免出现二义性。比如一些在 JAVA 语言中含有特殊含义的字符（换行符、回车符、单引号、双引号等）, 如果执行下面语句：

System.out.println(""");

JAVA 无法判断是输出 " , 还是出现 Bug, 多了一个 " 符号。所以使用转义 \" 表示输出为 " 

System.out.println("\"");

字符串转义字符

常用转义字符：

\\   反斜杠
\'   单引号
\"   双引号
\b   退格
\n   换行
\t   制表符, 相当于 Tab 键
\r   回车
\b   退格
\f   换页
\0   空字符

正则转义字符

$ ( ) * + - [ ] ? \ ^ { } |

对以上字符前加 \\ 即可进行转义。可以理解为对正则中的字符需要进行 \ 转义, 而在字符串中还需要对特殊字符再进行一次\ 转义

举例

读取本地文件, 通过 FastJson 读取 reviewText 中的数据。将其中的 ' 替代为 \'

{
    "reviewerID": "A11N155CW1UV02", 
    "asin": "B000H00VBQ", 
    "reviewerName": "AdrianaM", 
    "helpful": [0, 0], 
    "reviewText": "I had big expectations because I love English TV, in particular                 Investigative and detective stuff but this guy is really boring. It didn't appeal to me at        all.", 
    "overall": 2.0, 
    "summary": "A little bit boring for me", 
    "unixReviewTime":             1399075200,     
    "reviewTime": "05 3, 2014"
}
  1. replaceAll() 时正则替换, 其中' 不需要进行转义, 而对于 \ 需要进行转义 \\
  2. \ 和 ' 都是字符串中的特殊字符, 在字符串中需要再一次转义
  3. 在 \\' 每个符号前加上转义符则为 \\\\\'

整个代码段如下所示：

String encoding = "GBK";
String simple = new File("").getAbsolutePath() + File.separator + "data\\data.txt";
File file = new File(simple);
if (file.isFile() && file.exists()){
InputStreamReader reader = new InputStreamReader(new FileInputStream(file), encoding);
BufferedReader bufferedReader = new BufferedReader(reader);
String lineText = null;
String reviewText = null;
lineText = bufferedReader.readLine();
JSONObject review = JSONObject.parseObject(lineText);
reviewText = review.getString("reviewText").replaceAll("\'","\\\\\'");
System.out.println(reviewText);

结果为：

I had big expectations because I love English TV, in particular Investigative and detective stuff but this guy is really boring. It didn\'t appeal to me at all.
