在 JAVA 中执行 SQL 语句时遇到如下的问题

Exception in thread "main" java.sql.SQLSyntaxErrorException: 
You have an error in your SQL syntax; check the manual that 
corresponds to your MySQL server version for the right syntax 
to use near 'A50IGEA0XTJ99','B000W5KBI4','Gale A. Cleaves','[0,0]')' at line 1

报错提示 SQL 语法错误，执行语句的内容如下：

{
    "reviewerID": "A11N155CW1UV02", 
    "asin": "B000H00VBQ", 
    "reviewerName": "AdrianaM", 
    "helpful": [0, 0], 
    "reviewText": "I had big expectations because I love English TV, in particular                 Investigative and detective stuff but this guy is really boring. It didn't appeal to me at        all.", 
    "overall": 2.0, 
    "summary": "A little bit boring for me", 
    "unixReviewTime":             1399075200,     
    "reviewTime": "05 3, 2014"
}

只是通过 FastJson 读取 json 数据, 将其中的 value 插入到对应表中的 key 字段。 可以从上面看到在 reviewText 中字符串自身带有' 符号, 而在 SQL 语句中会从单引号出截断, 就造成了语法错误。

这就需要提到如果通过 SQL 语句插入 ' 符号的话, 可以使用符号 \' 替代, 比如插入上面的 reviewText 内容的话, 可以通过执行如下语句：

INSERT INTO Video (reviewText)
VALUES ('I had big expectations because I love English TV, in particular Investigative         and detective stuff but this guy is really boring. It didn\'t appeal to me at        all.')

结果为字段 reviewText 中插入内容为：

I had big expectations because I love English TV, in particular                 Investigative and detective stuff but this guy is really boring. It didn't appeal to me at        all.

但是如果不是通过直接在数据库或者相关的管理软件中执行如上语句的话, 比如是通过 JAVA 代码实现对数据库的增加操作, 则在代码中还需要再进行转义操作, 如下所示：

reviewText = review.getString("reviewText").replaceAll("\\'","\\\\'");
