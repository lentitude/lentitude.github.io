MySQL Access denied for user root@localhost 解决办法

Created2018-01-30|Updated2020-03-26|数据库
|Post View:

之前忘记密码之后, 按照 MySQL 忘记密码解决办法 重新修改了密码。 重启 MySQL 服务之后，使用 Navicat 可以正常连接数据库, 可以电脑重启之后, 使用 Navicat 连接数据库会提示

以下是解决该问题的步骤：

  1. 打开命令行, 使用命令 net start 查看 MySQL 是否启动。如果启动可以使用命令net stop mysql 停止MySQL的运行
  2. 在安装 MySQL 的 bin 目录下打开命令行, 执行 mysqld --skip-grant-tables 跳过权限检查
  3. 重开一个命令行窗口, 运行 mysql -u root , 显示如下：
  1. 进入 mysql 数据库
  1. 给 root 用户设置新密码
  1. 刷新数据库
  1. 退出 mysql
